# Quick Reader ⚡

**Fast. Immersive. Zen.**

Quick Reader is a premium, high-performance RSVP (Rapid Serial Visual Presentation) speed reading plugin for Obsidian. Designed for deep focus and maximum comprehension, it turns your Obsidian notes into a distraction-free, high-speed reading flow.

## 🚀 Experience the Zen
- **Dual-Tone Performance**: A high-contrast calibration using **Electric Blue** for UI and **Crimson Red** for the reading focus point.
- **Stable Anchor System**: Mathematical ORP (Optimal Recognition Point) centering that locks the focus letter to the exact center of the screen, eliminating eye jitter.
- **Intelligent Pacing**: Smart micropauses built into the logic for punctuation, structural headings, and callouts.
- **Zero Distraction**: Immersive glassmorphic modal that hides the rest of the vault to keep you in the flow.

## ⌨️ Desktop Shortcuts
- `Space`: Play / Pause (Start reading)
- `↑ / ↓`: Increase / Decrease Speed (+25 WPM)
- `← / →`: Seek (Jump 10 Words Backward / Forward)
- `C`: Toggle Context Expansion (View surrounding sentence)
- `Esc`: Close Zen Reader

## 🔩 Installation

### From Community Plugins (COMING SOON)
1. Open **Settings** → **Community plugins**
2. Browse and search for **"Quick Reader"**
3. Install and enable

### Manual Installation
1. Download `main.js`, `manifest.json`, and `styles.css` from the latest release.
2. Create folder `.obsidian/plugins/quick-reader/` in your vault.
3. Copy the files into that folder and reload Obsidian.

## 🛠️ Build from Source
1. Clone the repository.
2. Run `npm install` to load dependencies.
3. Run `npm run build` to generate the bundled plugin.
